
# Your Company — Static Website Starter

A clean, accessible, mobile-first static site you can customize quickly.

## What's inside
- `/index.html` (Home)
- `/about.html`, `/pricing.html`, `/contact.html`, `/privacy.html`, `/terms.html`
- `/css/styles.css` (modern, responsive CSS with variables)
- `/js/main.js` (nav toggle + footer year)
- `/assets/logo.svg`, `/assets/favicon.svg`
- `robots.txt`, `sitemap.xml`

## Customize
- Search and replace **Your Company** with your brand name.
- Update colors in `:root` variables inside `css/styles.css`.
- Replace placeholder emails and links in `footer` and `contact` page.

## Quick deploy
- **Netlify**: Drag & drop this folder to app.netlify.com.
- **Vercel**: `vercel` in the folder or import via dashboard.
- **GitHub Pages**: Push to a repo → Settings → Pages → deploy `/` root on `main`.
- **cPanel/FTP**: Upload all files to your web root (often `public_html`).

## SEO
- Update `og:image` and set your domain in `sitemap.xml` and `robots.txt`.
